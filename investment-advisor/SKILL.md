# 投资分析助手

## 系统概述

融合顶尖大师投资智慧的个人投资分析助手，帮助你做出更明智的投资决策。

**核心功能**：
- 📊 **大师框架分析**：八位大师思维框架（价值/逆向/宏观/情绪/反身性/成长/逆向/深度价值）
- 📈 **价格数据**：黄金、原油、股票实时价格
- 🌍 **宏观背景**：利率、通胀、GDP 等经济指标
- 🔍 **实时新闻**：市场动态 + 大师最新观点
- ✅ **事实核查**：对分析内容进行严格事实核查，避免幻觉信息
- ⚠️ **预警监控**：价格异动自动触发深度分析

## 快速使用

```bash
# 分析单个标的
python scripts/main.py --ticker GC=F --query "黄金为什么暴跌"

# 批量分析多个标的
python scripts/main.py --tickers GC=F CL=F --query "当前市场分析"

# 检查预警
python scripts/alert_monitor.py

# 交互式查询
python scripts/main.py --interactive
```

## 分析框架

### 🧭 大师智慧融合

系统内置八位投资大师思维框架，博采众长不止于一家之言：

**价值投资派**
- **巴菲特**：护城河、别人恐惧时贪婪、合理价格、能力圈
- **芒格**：逆向思考、多元思维模型、激励效应、确认偏见

**宏观对冲派**
- **达利欧**：宏观周期、风险平价、长期债务周期
- **索罗斯**：反身性理论、宏观交易、冒险原则

**成长投资派**
- **彼得·林奇**：成长股逻辑、身边调研、十倍股思维
- **约翰·邓普顿**：逆向投资、全球分散、估值纪律

**深度价值派**
- **霍华德·马克斯**：钟摆理论、第二层思维、周期与时机
- **迈克尔·伯里**：深度价值、另类数据、耐心等待

### ✅ 事实核查机制

金融领域幻觉信息可能致命。每次分析后，系统自动对关键事实陈述进行核查：
- ✅ 可能是正确的陈述（需要进一步验证）
- ⚠️ 模糊的陈述（缺乏具体数据或时间）
- ❌ 可能是错误的陈述（与已知事实不符）

### 🔍 多源新闻与研报

系统搜索多个来源进行交叉验证，不依赖单一信息源：

**大师观点来源**（8位大师）
- 巴菲特、芒格、达利欧、马克斯
- 索罗斯、彼得·林奇、邓普顿、迈克尔·伯里

**机构研报来源**
- 高盛（Goldman Sachs）、摩根士丹利（Morgan Stanley）
- 桥水（Bridgewater）、瑞银（UBS）、花旗（Citi）

**多源交叉验证**
- 📊 标记：多源证实的观点（不同来源一致）
- ⚠️ 标记：有分歧的观点（需要进一步判断）

### 📊 报告结构

```
1. 价格信号解读
2. 大师框架分析（八位大师视角）
   - 巴菲特视角（护城河/合理价格）
   - 芒格视角（逆向思考）
   - 达利欧视角（宏观周期）
   - 马克斯视角（钟摆理论）
   - 索罗斯视角（反身性）
   - 林奇视角（成长投资）
   - 邓普顿视角（逆向投资）
   - 伯里视角（深度价值）
3. 综合方向判断（短期/中期）
4. 风险提示
5. 投资建议
6. 事实核查结果
```

## 预警配置

编辑 `scripts/config.yaml` 自定义监控标的和阈值：

```yaml
watchlist:
  - name: 黄金
    ticker: GC=F
    alert_threshold: 0.03  # ±3%
```
